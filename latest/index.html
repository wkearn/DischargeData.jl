Abstract Quantities

# DischargeData.Quantity — Type.

A Quantity represents a time series of some quantity measured in a tidal channel. You might measure water levels (Stage) or flow (Discharge) or whatever water quality parameters you might think up (not implemented).

Each sample of a quantity is assumed to be drawn at a real-world time, so a Quantity is a vector of values of the quantity in some space T attached to a DateTime.

# DischargeData.@quantity — Macro.

@quantity name T

This macro simplifies the process of defining a quantity that conforms to the abstract specification.

Examples

@quantity Stage Float64

generates a new type Stage <: Quantity{Float64} which is a Real-valued time series
